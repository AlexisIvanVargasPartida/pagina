Egresado del Instituto Tecnólogico de Chihuahua, Ing. Adam Carrasco, desarrollador de software en "Oracle"



Desarrollo Web en el 2017 + Tips:

Iniciamos con dos definiciones escenciales, "Front-end" siendo todo lo que vemos por parte del navegador (HTML,CSS,JavaScript) y "Back-End" (Databases,BL,WS)


Tendencias en el desarrollo Web:


Web esta enfocado en 2 vertientes:


      Progressive Web Apps
      Single Page Applications(Respuestas muy rápidas)

Frameworks Back-End:


      Ruby On Rails
      DJango
      Laravel
      Spring Framework
      ASPNET

Back-End como servicio:


      Realm
      Firebase
      Socket.io
      Parse

En cuanto al Front-End, el framework más usado es Bootstrap, ya que el uso de este framework brinda ajustabilidad y es muy estable através de su patrón de diseños.


Preprocesadores CSS:


      less CSS
      SCCSS
      SASS

CSS proporciona herramientas de ámbito general para modificar varios elementos a gusto propio del HTML en general.


Frameworks JavaScript (2012-2017) populares:


      React
      Angular JS
      Vue JS
      JQuery

React esta escrito únicamente en JavaScript, es bueno y factible para editar ciertos componentes en la página web y no cada uno de ellos, sin alterar más elementos.


AngularJS tiene la ventaja de que su uso es en tiempo real, enlazando los componentes.


Vue acepta cualquier componente y es compatible con ellos.


Editores de texto más populares:


      Atom
      Sublime Text
      Visual Studio

"Functional Testing", hacer uso de testear aplicaciones es recomendable para medir su optimo funcionamiento.


Top 10 enseñanzas:


      Escoger un lenguaje de programación y hacerlo tu primer amor
      Adentrarse dentro de las estructuras de datos
      Entrenar habilidades para programar algoritmos (Hackerrank,CodeFights,LeetCode)
      Entender el funcionamiento completo de los frameworks y como funciona internamente
      Git y Github (Conocerlos y diferenciarlos)
      No pares en "es solo un proyecto escolar"
      No esperarse hasta 9° semestre para conseguir experiencia
      No hables,Actua
      Trata de crear tu propio negocio
      Leer mucho
      El ingles
